This project is my Academic Mini Project.

It is developed using Java programming language.

It uses several OOPs concepts such as 

Inheritance,Encapsulation,Polymorphism,Method Overloading,Method Overriding etc.

This project was developed in the Eclipse IDE.

This project serves as a software for the hospitals or vaccination centers which are involved in vaccination drive to store the information of the citizens who are eligible to get vaccinated
and also allocate them schedule place and time using Java Random class.
